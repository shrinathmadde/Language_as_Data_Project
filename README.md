# Language as Data - Assignment Scripts

## Quick Setup

```bash
# Create and activate virtual environment
python3 -m venv venv
source venv/bin/activate

# Install dependencies
pip install -r requirements.txt

# Install SpaCy German model
python -m spacy download de_core_news_md
```

## Run Scripts

```bash
# Run deliverable 3
python deliverable_3_corpus_statistics.py

# Run deliverable 4
python deliverable_4_tokenizer_analysis.py

# Run deliverable 5
python deliverable_5_dependency_parsing.py
```
